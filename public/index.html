What exactly is a Markov Chain?
To understand what a Markov chain is, first consider a problem that it is trying to solve. Given a set of states, and probabilities of transitioning from some state Si to some state Sj, can we predict the sequence that will likely come from some state Sk?

We can generate a possible sequence, by looking at potential transitions from Sk and make an educated guess at what the next state will be. An easy way to organize all of the potential transitions, is in a transition matrix. Here is an example of a transition matrix that only involves the states R, N and S.

In this matrix, the value at position i,j in the matrix is equal to the probability that given the current state is Si that the next state will be Sj. For example, half the time R will actually loop back to itself, which is why R,R has a value of 1/2. One fourth of the time, R will transition to N, so R,N has a value of 1/4. In order to generate a sequence, a starting state is picked, and then each consecutive state will be picked based on the probability given in the matrix.

What is great about this, is given a large number of sequences, we can create our own probability matrix by counting each transition between some Si and some Sj. For example, if we find that throughout all of our sequences, S1 goes to S2 5 times, S1 goes to S0 2 times and S1 goes to S1 3 times, then the probability of S1 going to S0, S1 and S2 is .2, .3 and .5 respectively. When we decide to generate a new sequence, if we are at S1 then the next state will be S0 one fifth of the time, S1 three tenths of the time, and S2 half the time.

These sequences can be created by any seqential data. In our example here, we are using tweets. Each state is a word in a tweet. We scan over hundreds of thousands of tweets to create this matrix that will tell us given the word "and", what are the likely words that Donald Trump will say next? This can be saved in a matrix as described above and used to create new sentences that will sound similar to the sentences that he has created himself!
